# Assets Directory

Place your project icon files here. These are used for Linux desktop shortcuts created by `install.sh`.

## Required files:
- `icon.png` - Icon for the **launch** desktop shortcut (recommended: 256x256 px)
- `icon_dev.png` - Icon for the **development** desktop shortcut (recommended: 256x256 px)

You can use any `.png` image. If no icons are provided, the desktop shortcuts will work but display without an icon.
